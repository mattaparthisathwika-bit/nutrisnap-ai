# AI Logs — 8x Engineer Contest Submission

This folder contains AI conversation logs as required by the 8x Engineer contest rules.

## How to Add Logs (Cursor)

After each Cursor session:
1. Open Chat panel → click `...` → **Export conversation**
2. Save the `.txt` file here as: `session-YYYY-MM-DD-HH.txt`
3. `git add ai-logs/ && git commit -m "log: add AI session log" && git push`
